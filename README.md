## Build via GitHub Actions (recommended — no local setup needed)

This repo is pre-configured with a GitHub Actions workflow.
Every push to `main` triggers an automatic build.

1. Go to the **Actions** tab of this repo
2. Click the latest workflow run
3. Download **app-debug-apk** from Artifacts

## Build locally

Requirements:
- JDK 17
- Android Studio Hedgehog (2023.1.1+) or Android SDK 35

### Android Studio
1. Open this folder in Android Studio
2. Wait for Gradle sync
3. Build → Build APK(s)
4. APK: `app/build/outputs/apk/debug/app-debug.apk`

### Command line
```bash
./gradlew assembleDebug
```

## Features
- Pure Android WebView (no Capacitor, no React Native)
- MutationObserver CSS class blocker
- Full cookie + DOM storage persistence
- File chooser / camera support
- Desktop Chrome user-agent spoof
- Back button navigation
- Cleartext HTTP support
